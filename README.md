# Portfolio PSFV - Federico Quiroga

Demo interactiva para postulación laboral orientada a gestión de calidad, seguimiento de avance, stock y reporting ejecutivo en proyectos solares fotovoltaicos.

## Publicación

Este repositorio está preparado para GitHub Pages. El sitio principal está en `index.html`.

## Nota

Los datos son simulados y generados para demo. La lógica representa un caso realista de seguimiento de un PSFV.
